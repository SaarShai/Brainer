# Example rubric — content output (edit to encode YOUR taste)

The score is only as sharp as this rubric. Replace the criteria below with the
specific things a reader bookmarks YOU for. A vague rubric ("is this good and
engaging") produces a vague score; a specific one produces a score you can trust.

Rate the candidate 0-5:

5 = all four criteria, cleanly; a reader would bookmark and implement it
4 = all four, minor verbosity or one soft spot
3 = hits 2-3 criteria; usable but forgettable
2 = hits 1; mostly generic
1 = hits none; correct-sounding and hollow
0 = blank, off-topic, hallucinated, or refused

Criteria:
1. Specific action — how to do something the reader can do tomorrow, not a
   vibe. Contains at least one copy-pasteable step / template / number.
2. Followable — anyone in the audience can act on it; no jargon wall.
3. Structured — replicable, step-by-step, not just inspirational.
4. Novel — the reader didn't already know this.

Meta-question (overrides the above): would someone bookmark this and come back
to implement it? If no, it's slop no matter how clean the prose — cap at 2.

Respond with exactly one digit 0-5 on the first line, then a one-line reason.
